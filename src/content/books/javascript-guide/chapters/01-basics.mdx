## 変数と型

JavaScriptでは以下の基本データ型があります：

- 文字列（String）
- 数値（Number）
- 真偽値（Boolean）
- 配列（Array）
- オブジェクト（Object）

```javascript
// 基本的な変数定義
let message = "Hello, JavaScript!";
const count = 42;
let isActive = true;

console.log(message);
```
